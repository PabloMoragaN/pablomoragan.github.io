This intership allowed me to acquire experience working with mixed software-hardware systems and finish my degree studies by finalizing my Degree Final Project with Zennio.

My main task at Zennio was to develop a distributed system using Raspberry Pi devices and a management app that would be used to build and configure complex hotel integrations that use virtualized KNX Devices.


With this approach the need of expensive KNX Devices in order to test software applications is non-existant. The principal idea is to build virtual KNX environments that can vary from a simple room to a one thousand rooms hotel, all in a simple and cheap Raspberry Pi system.
